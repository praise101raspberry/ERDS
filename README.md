# ERDS – Emergency Response and Dispatch System

## Overview

ERDS is a Computer-Aided Dispatch (CAD) system designed to support emergency response services such as Police, Emergency Medical Services (EMS), and Fire Departments.

The system enables emergency call handling, incident management, vehicle dispatch, responder tracking, and real-time monitoring.

This project is being developed as both a portfolio project and a large-scale software engineering project using Agile (Scrum) principles.
